## 🚀 Overview

A **modern, interactive Digital Wallet App** built with **Jetpack Compose**, designed to replicate a **real-world tactile card experience**.

✨ Store, scan, and manage:
- Credit/Debit Cards  
- Metro Cards  
- Driving Licenses  

With **smooth animations, ML-powered scanning, and intuitive UI.**

## ✨ Features

### 🎴 Interactive Card Stack
- Smooth expand/collapse animations  
- Realistic card stacking effect  
- Scroll-based transitions  

### 🔄 Card Interaction
- Tap → Expand card  
- Tap again → Flip card (front/back)  
- Haptic feedback for better UX  

### 📸 Smart Card Scanner
- Built with **CameraX + ML Kit**
- Detects:
  - Card Number  
  - Expiry Date  
  - Bank Name  
- **4-sec hold detection** for accurate capture  
- Uses real card image as background  

### 🧩 Drag & Drop Reordering
- Long press → reorder cards  
- Smooth gesture animations  

### 🌗 Dynamic Theming
- Auto adjusts text color based on background brightness
